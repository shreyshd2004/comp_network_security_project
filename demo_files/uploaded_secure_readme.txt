Welcome to the TFTP Demo!

This is a demonstration of the baseline and secure (AES-GCM with PSK) TFTP implementations.
TFTP (Trivial File Transfer Protocol) is a simple protocol for transferring files over UDP.

Features demonstrated:
- File downloads (GET)
- File uploads (PUT)
- Error handling
- Binary file transfers
- Secure mode with authenticated encryption (AES-GCM)
